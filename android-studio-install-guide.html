Android Studio 설치

안드로이드 스튜디오 설치 방법 완전 정리

카테고리: 앱 개발 입문 · 작성일: 2026-04-08

안드로이드 앱을 직접 만들어 보고 싶다면 가장 먼저 설치해야 하는 대표 도구가 바로 Android Studio입니다.
하지만 처음 시작하는 사람에게는 설치 과정이 생각보다 어렵게 느껴질 수 있습니다.

“어디서 다운로드해야 하나요?”, “설치 후 뭐를 더 깔아야 하나요?”,
“SDK, 에뮬레이터, JDK는 또 뭔가요?” 같은 부분에서 막히는 경우가 많습니다.

이 글에서는 완전 초보자 기준으로 Android Studio를 설치하는 방법부터, 설치 후 꼭 확인해야 하는 기본 세팅까지 최대한 쉽게
순서대로 정리해 보겠습니다.

1. Android Studio가 무엇인가요?

Android Studio는 안드로이드 앱을 만들기 위한 공식 개발 프로그램입니다.

쉽게 말하면, 앱 화면을 만들고, 버튼을 넣고, 기능을 작성하고, 테스트하고, 최종적으로 앱 파일을 만드는 데 필요한 거의 모든 기능이
들어 있는 도구입니다.

보통 아래와 같은 작업을 할 수 있습니다.

  • 안드로이드 앱 화면 만들기
  • Kotlin 또는 Java 코드 작성
  • 버튼, 텍스트, 이미지 배치
  • 가상 스마트폰에서 앱 테스트
  • APK / AAB 앱 파일 생성
안드로이드 앱을 제대로 만들고 싶다면 결국 Android Studio를 다루게 되는 경우가 매우 많습니다.

2. 설치 전에 먼저 확인할 것

설치 전에 가장 먼저 확인해야 하는 것은 내 컴퓨터 사양이 너무 낮지 않은지입니다.

Android Studio는 일반 문서 프로그램보다 훨씬 무거운 편이기 때문에, 너무 오래된 PC나 램이 적은 컴퓨터에서는 느리게 동작할 수
있습니다.

권장 체크 항목

  • Windows 10 / 11 또는 macOS 사용 여부
  • 램 8GB 이상 권장 (가능하면 16GB가 더 편함)
  • 저장 공간 여유 20GB 이상 확보
  • 가상폰 실행을 위한 CPU 가상화 기능 지원 여부

특히 저장 공간은 생각보다 많이 필요합니다. Android Studio 본체뿐 아니라 SDK, 에뮬레이터, 시스템 이미지까지 설치되기 때문입니다.

저장 공간이 부족하면 설치는 되더라도 나중에 에뮬레이터 생성이나 SDK 업데이트에서 자주 막힐 수 있습니다.

3. Android Studio 다운로드 방법

Android Studio는 반드시 공식 사이트에서 다운로드하는 것이 좋습니다.

검색해서 나오는 이상한 다운로드 사이트나 오래된 블로그 링크를 따라가면 구버전 파일을 받거나 잘못된 설치 파일을 받을 수 있습니다.

다운로드 순서

  1. 공식 Android Studio 다운로드 페이지에 접속합니다.
  2. 운영체제(Windows / Mac)에 맞는 설치 파일을 선택합니다.
  3. 이용 약관 체크 후 다운로드를 시작합니다.

다운로드 파일 크기가 꽤 크기 때문에 인터넷 속도에 따라 시간이 조금 걸릴 수 있습니다.

4. 설치 과정 따라하기

설치 파일을 실행하면 기본 설치 마법사가 시작됩니다. 대부분은 크게 어렵지 않으며, 처음에는 기본값으로 진행해도 무방합니다.

기본 설치 흐름

  1. 설치 파일 실행
  2. Next 클릭
  3. Android Studio / Android Virtual Device 선택 확인
  4. 설치 경로 확인
  5. Install 진행

보통 초보자는 설치 경로를 굳이 복잡하게 바꾸지 않고 기본 경로로 설치하는 것이 가장 안정적입니다.

설치가 끝나면 Android Studio를 처음 실행하게 됩니다.

5. SDK와 필수 구성 요소 설치

Android Studio를 설치했다고 해서 바로 앱 개발이 완전히 준비되는 것은 아닙니다.

실제 앱을 만들고 실행하려면 Android SDK 같은 추가 구성 요소가 함께 설치되어야 합니다.

쉽게 이해하면

  • Android Studio = 작업실
  • SDK = 안드로이드 앱 개발 도구 세트
  • 에뮬레이터 = 가상 스마트폰

처음 실행 시 Setup Wizard가 열리면 보통 필요한 구성 요소를 자동으로 설치할 수 있도록 안내해 줍니다.

처음에는 보통 이렇게 진행하면 됩니다

  1. Standard 설치 선택
  2. 필요한 SDK 자동 다운로드 진행
  3. 라이선스 동의
  4. 설치 완료까지 기다리기
처음 설치 시 시간이 가장 오래 걸리는 구간이 바로 SDK와 시스템 이미지 다운로드입니다.

6. 에뮬레이터(가상폰) 설정 방법

앱을 테스트하려면 실제 안드로이드 폰을 연결하거나, 컴퓨터 안에서 가상 스마트폰을 실행해야 합니다.

이 가상 스마트폰 기능을 에뮬레이터라고 부릅니다.

에뮬레이터가 필요한 이유

  • 실제 폰이 없어도 앱 테스트 가능
  • 다양한 화면 크기 확인 가능
  • 앱 실행 오류를 빠르게 확인 가능

설정 흐름

  1. Android Studio 실행
  2. Device Manager 열기
  3. Create Device 선택
  4. 원하는 기기 모델 선택
  5. Android 버전(시스템 이미지) 선택
  6. Finish로 생성 완료

처음에는 너무 무거운 고사양 기기보다는 일반적인 Pixel 계열 기본 설정으로 시작하는 것이 편합니다.

에뮬레이터가 너무 느리다면 컴퓨터 램 부족, 저장 공간 부족, CPU 가상화 비활성화 문제일 가능성이 큽니다.

7. 첫 프로젝트 만들기

설치가 끝났다면 이제 실제로 앱 프로젝트를 하나 만들어 볼 수 있습니다.

기본 흐름

  1. New Project 클릭
  2. Empty Activity 선택
  3. 앱 이름 입력
  4. 언어는 Kotlin 선택
  5. Minimum SDK 선택
  6. Finish 클릭

초보자라면 언어는 Kotlin으로 시작하는 것을 추천합니다.

현재 안드로이드 개발에서는 Kotlin이 가장 많이 권장되는 언어 중 하나이며, 공식 예제도 Kotlin 기준이 많은 편입니다.

8. 앱 실행 테스트하기

프로젝트를 만들었다면 이제 실제로 앱이 실행되는지 확인해 봐야 합니다.

실행 방법

  1. 상단 실행 버튼(▶) 클릭
  2. 실행할 기기 선택
  3. 에뮬레이터 또는 실제 폰 선택
  4. 앱 빌드 및 실행 대기

첫 실행은 시간이 꽤 걸릴 수 있습니다. Gradle 빌드, SDK 연결, 초기 세팅이 함께 이뤄지기 때문입니다.

하지만 첫 실행만 넘기면 이후에는 훨씬 수월하게 테스트할 수 있습니다.

처음 실행에서 오래 걸린다고 해서 바로 오류라고 단정하지 말고 몇 분 정도 기다려 보는 것이 좋습니다.

9. 설치 중 자주 생기는 오류

Android Studio 설치는 비교적 자동화가 잘 되어 있지만, 초보자가 자주 막히는 대표 문제들이 있습니다.

자주 발생하는 문제

  • SDK 다운로드 실패
  • 에뮬레이터 실행 안 됨
  • Gradle 동기화 오류
  • 램 부족으로 프로그램이 너무 느림
  • 가상화(VT-x / AMD-V) 비활성화

가장 많이 막히는 원인

가장 흔한 것은 인터넷 문제, 저장 공간 부족, 가상화 설정 문제입니다.

특히 에뮬레이터가 아예 실행되지 않거나 엄청 느리게 뜨는 경우는 BIOS에서 CPU 가상화 기능이 꺼져 있는 경우가 많습니다.

설치가 안 될 때는 무작정 재설치하기보다 어떤 단계에서 막혔는지 먼저 확인하는 것이 훨씬 중요합니다.

정리: 초보자가 가장 먼저 해야 할 것

Android Studio를 처음 시작하는 초보자라면 아래 순서대로 진행하면 가장 무난합니다.

  1. 공식 사이트에서 Android Studio 다운로드
  2. 기본값으로 설치 진행
  3. SDK 자동 설치 완료
  4. 에뮬레이터 1개 생성
  5. Empty Activity 프로젝트 생성
  6. 앱 실행 버튼 눌러 테스트

처음에는 복잡해 보일 수 있지만, 사실 가장 어려운 구간은 “처음 설치” 한 번뿐인 경우가 많습니다.

Android Studio 설치만 끝나도 앱 개발의 진입장벽 절반 이상은 넘었다고 봐도 됩니다.

자주 묻는 질문

Q1. Android Studio는 무료인가요?

네, 무료입니다. 안드로이드 앱 개발을 위해 누구나 설치해서 사용할 수 있습니다.

Q2. Android Studio만 설치하면 바로 앱을 만들 수 있나요?

대부분은 가능합니다. 다만 처음 실행 시 SDK, 에뮬레이터 같은 추가 구성 요소 설치가 필요할 수 있습니다.

Q3. 초보자는 Java와 Kotlin 중 무엇으로 시작해야 하나요?

보통은 Kotlin을 추천합니다. 현재 안드로이드 개발에서 더 많이 권장되는 흐름입니다.

Q4. 실제 스마트폰 없이도 앱 테스트가 가능한가요?

네, 가능합니다. Android Studio의 에뮬레이터 기능을 사용하면 컴퓨터 안에서 가상 스마트폰을 실행할 수 있습니다.

Q5. 설치 후 제일 먼저 뭘 해야 하나요?

첫 프로젝트를 하나 만들고, 앱이 실행되는지 확인하는 것이 가장 좋습니다. 실행까지 성공하면 이후 학습이 훨씬 쉬워집니다.
